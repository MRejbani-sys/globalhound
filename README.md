# 🐾 GlobalHound

**Your Passport to Global Opportunity**

A subscription-based intelligence platform for global relocation and visa-friendly employment. This is a fully functional HTML/CSS/JS prototype ready for Vercel deployment.

## Features

- 🌍 **Landing Page** — Hero, pillars, pricing, testimonials, FAQ
- 💼 **Jobs Portal** — 10 visa-friendly job listings with live filters
- 🔒 **GCC Private Market** — Exclusive Middle East listings with package details
- 📊 **Country Intelligence Hub** — 12 country profiles with metrics
- ⊞ **Members Dashboard** — Weekly Drop, Referral Rewards, Activity Feed
- 🔐 **Auth System** — Login & Register (localStorage-based simulation)

## Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Click **Deploy** — no build settings needed

## Tech Stack

- Pure HTML, CSS, JavaScript — zero dependencies
- Google Fonts (Playfair Display + DM Sans)
- Fully responsive (mobile, tablet, desktop)
- localStorage-based simulated auth
